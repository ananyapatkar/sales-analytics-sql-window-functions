# Sales Analytics using SQL Window Functions

This project performs sales analytics using PostgreSQL on a Superstore dataset.
It demonstrates the use of SQL window functions, ranking techniques, and time-series analysis to extract meaningful business insights.

## Key Features

- Data cleaning and date normalization
- Aggregation using GROUP BY
- Ranking customers using ROW_NUMBER, RANK, and DENSE_RANK
- Running total sales analysis
- Month-over-Month (MoM) growth calculation using LAG
- Top-N product analysis using CTEs
- Exporting analytical outputs to CSV files

## Tech Stack

- PostgreSQL
- SQL (Window Functions)
- DBeaver (for query execution and exports)
